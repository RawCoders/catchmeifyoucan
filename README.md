# Catch Me If You Can

A multiplayer game where you are spawned with real players and NPCs and all players look identical, you have to find yourself without being detected and find others by observing their movements.

## Tools and Frameworks

- [Phaser 3](https://phaser.io) - Game framework
- [Tiled](https://www.mapeditor.org/) for drawing maps
- [Zelda-like Tilesets and Sprites](https://opengameart.org/content/zelda-like-tilesets-and-sprites) for map sprites
- [Craftpix Free Sprites](https://craftpix.net/freebies/free-warrior-4-direction-character-sprites/) for character sprites
- [Firebase](https://firebase.google.com) for realtime sync of game data
